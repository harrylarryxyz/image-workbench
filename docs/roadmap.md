- Provider profile list and environment fallback.

- Task list and Gallery metadata.
- Route diagnostics and error classification.

- Mask upload and Konva mask editor.
- Reuse generated images as references.

## Phase 4

- Prompt library and prompt enhancer.
- Style templates and prompt history.

## Phase 5

- React Flow canvas projects.
- Prompt/Image/Task nodes.
- Reference edges and canvas export/import.
